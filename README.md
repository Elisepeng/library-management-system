# 图书馆管理系统

## 项目简介

本项目是一个基于 Spring Boot + MyBatis 开发的图书馆管理系统，实现了图书管理、用户管理、图书借阅、归还等功能。

该项目主要用于学习 Java Web 后端开发流程，加深对 Spring Boot、MyBatis 与 MySQL 的理解。


## 技术栈

- Java
- Spring Boot
- MyBatis
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Maven

## 功能模块

### 用户功能

- 用户登录
- 图书查询
- 借阅图书
- 归还图书

### 管理员功能

- 图书管理
- 用户管理
- 借阅记录管理

## 项目特点

- 基于 Spring Boot 搭建后端
- 使用 MyBatis 操作数据库
- MySQL 存储数据
- 前后端通过 REST API 进行交互
